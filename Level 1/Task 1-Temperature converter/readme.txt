# Temperature Converter

## Overview
The Temperature Converter is a simple Java program that converts temperatures between Fahrenheit and Celsius based on user input.

## Features
- Input a temperature value.
- Specify the unit of measurement (F for Fahrenheit and C for Celsius).
- Converts the temperature to the other unit of measurement.
- Displays the converted temperature.

## How to Use
1. Run the program.
2. Enter the temperature value when prompted.
3. Enter the unit of measurement (F for Fahrenheit and C for Celsius) when prompted.
4. The program will display the converted temperature in the other unit of measurement.

## Example
Enter temperature value: 100
Enter unit of measurement (F for Fahrenheit and C for Celsius): F
Converted temperature is 37.77777777777778 Celsius.
