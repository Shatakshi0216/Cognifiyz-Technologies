# Random Password Generator

## Overview
The Random Password Generator is a simple Java program that generates a random password based on user-specified length. The password includes uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Input the desired length of the password.
- Generates a random password containing a mix of uppercase letters, lowercase letters, numbers, and special characters.
- Displays the generated password.

## How to Use
1. Run the program.
2. Enter the desired length of the password when prompted.
3. The program will display the generated random password.

## Example
nter the length of password: 12
Generated password is: aB3#dFg$1K2!
